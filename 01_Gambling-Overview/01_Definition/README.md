# Spiel-Tracker

Diese statische Webseite kann kostenlos auf GitHub Pages, Netlify oder Vercel gehostet werden.

## So veröffentlichen

1. Lade den Ordner in ein GitHub-Repository hoch.
2. Aktiviere GitHub Pages für den Hauptbranch.
3. Öffne die veröffentlichte URL.

Die Daten werden im Browser des Nutzers gespeichert (Local Storage).
